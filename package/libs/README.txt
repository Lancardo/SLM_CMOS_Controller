IMPORTANT: Place your local gxipy SDK files in this directory.

This should include:
- The gxipy Python module files
- Any required DLL files
- Any other dependencies required by the gxipy SDK

The application expects to find the gxipy module in this directory when running the packaged executable.

If you need to install the SDK separately, please follow the manufacturer's instructions and ensure
the SDK is properly registered in your system.
